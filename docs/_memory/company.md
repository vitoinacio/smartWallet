# Perfil

**Nome:** SmartWallet
**Setor:** Finanças Pessoais
**Linguagem de saída:** pt-BR
**Usuário de teste:** teste@gmail.com (mock interativo)

## Stack
- React 19, TypeScript, Vite 6, Tailwind CSS 3.4
- shadcn/ui (Radix primitives)
- Sonner (toasts), Recharts (gráficos)
- Vitest (unit tests), Playwright (E2E)
- GitHub Actions (CI), Vercel (deploy)

## Funcionalidades Implementadas
- Auth (Login/Cadastro) com animações
- Dashboard com resumo financeiro + gráficos
- Gestão Financeira com CRUD + filtros + orçamentos + transações recorrentes
- Dark mode
- Mock interativo para demonstração
- Testes automatizados (33 unit + 9 E2E)
- CI/CD pipeline
- 10 templates de recorrência (Netflix, Spotify, Aluguel, etc)
